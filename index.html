Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The class attribute works in a similar way to the id attribute.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Enter > git status to check which branch your in. Want to be in 'main'.
  • If not, then enter > git checkout main
  • Pull down the latest main branch by entering > git pull origin main
  • Create a new branch using >git checkout -b "new-branch" (without the quotes) where new-branch is whatever you want to name the branch
  • Confirm you're in new branch by again entering > git status
  • You can now start making changes to the checked out code.
